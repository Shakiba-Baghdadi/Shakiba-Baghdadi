# Real-Time Biosignal Processing & HMI Research

Engineering repository focused on biomedical signal processing, real-time human-machine interfaces (HMI), and applied machine learning for healthcare applications.

### 🛠️ Technical Stack & Skills
- **Core Areas:** Biosignal Analysis (EOG, EEG), Real-time Feature Extraction & Classification.
- **Tools:** Python (Scikit-Learn, TensorFlow/PyTorch), MATLAB, Simulink, System Integration.

---

## 🚀 Featured Project: Real-Time EOG-Based HMI System
Development of an end-to-end electrooculography (EOG) system for real-time human-machine system control, graphical menu navigation, and 2D object manipulation.

- **Offline Classification Accuracy:** 99.8% (UNIOVI_EOG Dataset)
- **Real-Time 4-Class Movement Accuracy:** 97%
- **Real-Time 9-Class Movement Accuracy:** 87%
